# v0.3 Combat Design Reference

This note freezes the qualitative move structure for future v0.3 combat work.
It records authored design decisions only; it does not define exact numerical
balance or implementation formulas.

## Shared Kit Structure

Each Drifter should ultimately have:

- four standard moves
- one super attack

This does not yet define unlock order, exact balance, or which moves are
available at the beginning of the game.

## Qualitative Scales

Physical output:

- None
- Low
- Medium
- High
- Massive

Magic output:

- None
- Low
- Medium
- High
- Massive

Accuracy:

- Low
- Medium
- High
- Guaranteed

Resource:

- None
- Mana
- Character Resource
- Super

## Joruun Veyr Reference Kit

Joruun Veyr is the neutral reference character for v0.3 combat calibration.

| Move | Physical | Magic | Accuracy | Resource |
| --- | --- | --- | --- | --- |
| Bring the Horse to Water | Medium | None | High | None |
| Lightning Palm | Low | High | Low | Mana |
| Tempest Surge | None | Medium | Medium | Mana |
| Hydro Whip | None | Low | Medium | Mana |
| Coagulated Torrent | None | Massive | Guaranteed | Super |

## Roster Calibration

Branoc:

- Physical output exceeds Joruun.
- Magic output is far below Joruun.

Azhvielle:

- Magic output exceeds Joruun.
- Physical output is below Joruun.

Zhaivra:

- Accuracy exceeds Joruun.
- Raw output depends more heavily on compounds and setup.

Joruun:

- Serves as the neutral reference point.
- Establishes what Low, Medium, High, and Massive mean relative to the other
  Drifters.
- One complete character kit should be made operational and validated before the
  remaining characters are fully implemented.
